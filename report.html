CSS
导航设计
由于阿根廷国旗蓝白相间的设计，网站也设计成以蓝色和白色位主题色彩。其中导航背景为淡蓝，字体为白色，采用流动盒
子布局

: flexdisplay ;
: top 0;
: background-color rgb(138,202,240);

: width 100%;
: position fixed;
}

问题与解决
制作网页的过程中，我遇到的第一个问题是position：fixed，为了使导航，目录，按钮，背景图片不随滚轮移动，我设置
了以上部分的绝对位置，但由于起初设计页面时并未有条理地规划，导致我设置fixed属性时使相邻的div发生了错位，导致
内容被遮挡。所以，在后续的制作中我首先实现需要固定的部分，调整好导航，背景图片，目录，按钮的位置后再来填充主
要内容
第二个问题出现在目录的制作，由于页面的内容过多，我需要用目录来跳转到指定位置。 最开始我选择直接链接跳转到id
以实现这个功能，但在验证时发现这种方法跳转的位置并不准确，后来由于时间规划不充分以及学习js到位，还是未能解决
这个问题。
最后一个问题是由于我的页面需要许多的图片，所以加载的时间会略长，如果跳转地比较频繁，就会出现明显地卡顿。
实验总结
此次地实验涉及到HTML，CSS，响应式设计，完成网页后我巩固了web开发的基础知识，明白了网页开发的设计创意和实
现本身都至关重要。制作一个网页首先应该规划好布局，选择何种设计创意，制定好颜色基调，以及各个部分之间的逻辑联
系，写html时，每个区块属性中的class或id尽量直接易懂，以方便后续css部分的处理。然后是css,为了使页面美观，生
动，需要从很多资料和网页上学习，好的实例值得借鉴。最后是网页的优化，包括图形的压缩，代码的优化，js的使用。总
之，HTML和CSS是构建网页的基础。我将继续学习学习网页开发的知识，深入学习和实践。
