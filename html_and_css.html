HTML & CSS 学习总结
学习HTML和CSS目的是为了能够创建并设计网页，到目前为止，我已经学习了HTML和CSS的基础知识了解了网页的基本
结构。现在我已经利用HTML构建网页的基本结构，使用标题，段落，图片，链接这些元素，可以利用CSS美化文字，图片
这样的单个元素。在学习HTML和CSS的过程中，也常常遇到困难，比如：当嵌套的区块元素多了以后且有一些用的类选择
器，有一些用的是id选择器时而有一些都没有，写CSS的话就容易混淆导致内层的样式不生效，这时就要逐层检查选择器是
否写正确。当写实际当宽度和高度这样的属性时可以结合浏览器的检查元素样式进行编辑，这样可以更准确地达到预期地效
果。学习过程中，我明白了制作一个网页首先应该构思好网页的结构，网页不只是二维的，还可以是三维的。所以首先设计
背景，然后是首部尾部导航，以及一些位置固定的元素比如目录，还有按钮。接下来就可以着力制作主题内容。网页额布局
由多种，同行的可以选择flex-box，多行的可以选择网格布局。另外就是响应式设计，比如光标的变换，按钮的按压效果，
还有显示窗口大小变化后网页中的布局也要随之改变不能影响阅读。学习过程中不足之处就是实践太少，学到的知识不能及
时反馈给自己。接下来，我会去了解更多有关于HTML和CSS的知识，深入学习web开发的技术。
